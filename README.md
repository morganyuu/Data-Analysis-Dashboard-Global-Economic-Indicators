# Data-Analysis-Dashboard-Global-Economic-Indicators
An interactive Tableau dashboard analyzing macroeconomic indicators for 200+ countries from 2010 to 2025 using a dataset sourced from Kaggle. Users can dynamically select economic statistics via filters to explore trends on a global map and accompanying charts, with color-coded visualizations and legends for clear insights.


Dataset sourced from Kaggle: https://www.kaggle.com/datasets/tanishksharma9905/global-economic-indicators-20102025
---

## Included Indicators

- Inflation (Consumer Price Index %)
- GDP (Current USD)
- GDP per Capita
- GDP Growth (% Annual)
- Unemployment Rate
- Real Interest Rate
- Public Debt (% of GDP)
- Government Expense and Revenue
- Current Account Balance
- Gross National Income
- Tax Revenue

---

## Dashboard Features

- **Interactive map** showing each country with selected economic statistics.
- **Graph** to visualize economic trends over time.
- **Dynamic filters** using **parameters** and **calculated fields** to explore specific indicators.
- Color-coded values with legends for intuitive interpretation.
- Easy comparison between countries and years.
- **Summary overview** displaying key economic factors at a glance for quick insights.
